# DocuMind Design System

## Brand personality: Intelligent, clean, trustworthy
## Color palette
- Primary: #6C5CE7 (purple)
- Secondary: #00B894 (teal)
- Background: #F8F9FA / Dark: #1A1A2E
- Text: #2D3436 / Dark: #EAEAEA

## Typography
- Headings: Inter, 600/700 weight
- Body: Inter, 400 weight
- Mono (code/filenames): JetBrains Mono

## Component conventions
- All components in `frontend/src/components/`
- Pages in `frontend/src/pages/`
- Use shadcn/ui as the base component library
- Tailwind CSS for all styling — no inline styles, no CSS modules
